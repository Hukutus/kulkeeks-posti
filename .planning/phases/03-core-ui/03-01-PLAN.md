  <action>
Create `src/lib/delivery-utils.ts` with four exported functions following the exact patterns from 03-RESEARCH.md Pattern 3:

1. `getTodayISO()`: Returns `YYYY-MM-DD` using local date methods (`getFullYear`, `getMonth`, `getDate`) — NOT `toISOString()` which returns UTC.

2. `getCurrentWeekISO()`: Returns 7 ISO date strings from Monday to Sunday. Uses `(dayOfWeek + 6) % 7` formula to convert JS Sunday=0 to Monday-indexed offset. Sets hours to 0,0,0,0 on the Monday anchor.

3. `filterWeekDeliveries(deliveryDates: string[], weekISO: string[])`: Creates a Set from weekISO, filters deliveryDates against it.

4. `isDeliveryDay(deliveryDates: string[], todayISO: string)`: Simple `includes` check.

3. Add `caveat.variable` to the `<html>` element's className: `<html lang={locale} className={caveat.variable}>`
4. Do NOT specify explicit weights — Caveat is a variable font, next/font handles it automatically
5. Keep all existing layout.tsx code intact (NextIntlClientProvider, hasLocale check, setRequestLocale, generateStaticParams)

**CSS theme (globals.css):**
Replace the contents of globals.css with:
```css
@import "tailwindcss";

@theme inline {
  --font-handwriting: var(--font-caveat);
}
```

IMPORTANT: Use `@theme inline` (not bare `@theme`) because `--font-caveat` is a CSS variable set at runtime by next/font. Bare `@theme` resolves at build time and would lose the `var()` reference.
